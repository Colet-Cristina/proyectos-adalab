# "De Cero a Code: Mi Viaje en Adalab"

¡Bienvenida a mi repositorio de ejercicios!
Aquí voy documentando todo mi proceso de aprendizaje en el **Bootcamp de Programación Web de Adalab**. 👩‍💻

En este espacio recopilo los retos, ejercicios prácticos y proyectos que voy desarrollando.

---

## 📚 Estructura de Aprendizaje

El repositorio está organizado por módulos, siguiendo el programa del bootcamp:

- **Modulo-1-HTML-CSS**: Bases de la maquetación web, Flexbox, Grid y diseño responsivo.
- **Modulo-2-JavaScript**: Lógica de programación, manipulación del DOM, consumo de APIs y almacenamiento local.
- **Modulo-3-React**: Desarrollo de interfaces dinámicas y componentes reutilizables.
- **Modulo-4-Express-BD**: Desarrollo de servidores, APIs REST y bases de datos.

---

## 🛠 Tecnologías Principales

- **Frontend**: HTML5, CSS3, JavaScript (ES6+), React.
- **Backend**: Node.js, Express.
- **Herramientas**: Git, GitHub, Terminal.
